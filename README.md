# Threat-Tracker
Automated Vulnerability & Log Analysis Tool

ThreatTracker is a cybersecurity automation project that:
- Scans for open ports
- Detects failed login attempts
- Generates a security report in JSON format

# Skills Demonstrated
- Python scripting for cybersecurity
- Log analysis & pattern recognition
- Network scanning fundamentals
- Automation & reporting

# How to Run
```bash
git clone https://github.com/gandhlerose/ThreatTracker.git
cd ThreatTracker
python3 threattracker.py
